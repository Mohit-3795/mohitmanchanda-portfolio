01
In production · THEY internal
For B Corp recertification

B Corp Recertification Dashboard

Tracks our certification readiness across the 5 stakeholder areas (Workers, Customers, Environment, Community, Governance) plus the 7 new B Lab foundation requirements. Built because spreadsheets stopped scaling.

The dashboard auto-calculates a weighted readiness score from live evidence data, flags high-risk gaps, and logs every admin change. Leadership now checks status in real time instead of waiting for weekly slide updates.

  • Weighted readiness engine from live evidence data
  • Role-based auth (admin / viewer) with JWT cookies
  • Audit log on every write with before/after diffs
  • Emissions tracking for carbon-neutral status

02
In production · Daily use
For leadership presentations

Motherson Slide Library

Hundreds of leadership presentation slides — built over years for the Chairman’s office and investor decks — sitting in folders nobody could search through.

This tool extracts every slide from .pptx files, indexes them visually, and serves a searchable dashboard. Finding the right slide went from 30 minutes of digging to seconds. The team uses it daily when preparing new decks.

  • Auto-extraction from .pptx files using JSZip
  • Visual thumbnail grid grouped by presentation
  • Keyword search across all slide content
  • Zero-setup launch — double-click to run locally

03
In production · Vision 2030
For Motherson’s Vision 2030

Motherson Story Tracker

To support Motherson’s Vision 2030 ambitions we needed to collect compelling stories from across 12 business divisions (Wiring Harness, Vision Systems, Aerospace, and more) and 5 global regions, mapped to 16 storytelling themes — Automation, Digital Twin, AI, Operational Excellence, Sustainability, and others.

Spreadsheets couldn’t show gaps. This tool can. It surfaces which theme-division combinations lack coverage, lets editors tag stories across themes, and includes an OpenAI-powered chatbot that analyses patterns across hundreds of submissions.

  • Interactive story roadmap across themes × divisions
  • Gap analysis showing missing narratives instantly
  • AI assistant (OpenAI) for editorial pattern detection
  • Excel I/O to import existing data and export for review
  • Secure JWT auth with per-user accounts

04
Personal project · Claude Skill
A Claude Skill I authored

Vastu Pandit

A standing AI advisor that reads a decision through three traditional lenses — Jyotish, Vastu, and numerology — and answers like a trusted family pandit: a clear one-line verdict, and a remedy or better date when something isn’t auspicious.

Built as a proper Claude Agent Skill — routing logic, 14 reference playbooks, and Python scripts that actually compute the panchang and birth chart (pyephem) rather than guess. It onboards a new user, writes their profile, and personalises every answer after that.

  • Real computation — panchang & kundli via Python + pyephem, never from memory
  • Profile-based — onboards once, then tailors to the chart
  • Mentor-style answers — one-line verdict + remedy, not a jargon dump
  • Shareable — swap in a blank profile to hand it to anyone
